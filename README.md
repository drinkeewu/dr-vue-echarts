**A Echart Library for Vue.js** 📈📊

[使用说明文档](https://drinkeewu.github.io/dr-vue-echarts/)

## Install
```node
npm install dr-vue-echarts
```

## Quick Start
```js
import Vue from 'vue';
import DrVueEcharts from 'dr-vue-echarts';

Vue.use(DrVueEcharts)
```
